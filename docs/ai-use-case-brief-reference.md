# AI Use Case 참고문서

이 문서는 내부 공유용 PPT와 결과서 문구를 정리할 때 기준으로 삼는 원문 참고자료입니다. 발표자료를 만들 때는 아래 원문의 고객 Pain Point, 핵심 기능, 사용 목적, 기대 효과를 우선 반영합니다.

## 발표 성격

- 대상 독자는 내부 공유 참석자입니다.
- 주제는 `AX Use Case`이며, 핵심은 AI를 이용한 프로젝트 과제를 수행했다는 점을 짧고 분명하게 보여주는 것입니다.
- 발표자료는 실제 운영 전환, 고객 제안, 상용 도입을 설득하는 문서가 아니라 팀 내부 공유용 과제 결과 보고에 가깝습니다.
- 따라서 본문은 `무엇을 만들었는가`, `어디에 AI를 썼는가`, `Use Case 원문의 Pain Point와 어떻게 연결되는가`, `화면으로 어떤 결과가 보이는가`를 중심으로 구성합니다.
- 한계나 보완점은 필요할 때만 짧게 다루고, 발표의 중심 메시지로 두지 않습니다.

## 발표자료 해석 기준

- 중심 메시지는 `AI 커밋 분석 기반 프로젝트 자원 관리 서비스`입니다.
- 발표 흐름은 고객 Pain Point를 먼저 보여주고, 실제 앱 화면으로 그 Pain Point를 어떻게 다루는지 증명합니다.
- Project Chat, GraphRAG, AI Code Review, AI Progress, Dashboard/Risk/Resource 지표는 각각 독립 기능 목록이 아니라 하나의 프로젝트 관리 Use Case 흐름으로 연결합니다.
- 적용 범위나 보완점은 본문 메시지를 약하게 만드는 전면 주제가 아니라, 마지막 장에서 PoC 적용 구분과 함께 짧게 정리합니다.
- 원문에 있는 정량 가치인 `프로젝트당 1MM 이상 예상`은 근거 없는 확정 성과처럼 쓰지 않고, PoC 기대 효과 또는 추정 효과로 표현합니다.

## 원문

```text
AX Use Case
Use case 명: AI 커밋 분석 기반 프로젝트 자원 관리 서비스
Use case 설명: AI를 통해 산출물 및 커밋된 내용을 분석하여 프로젝트 개발진척의 정량적인 분석을 수행하고, 이를 바탕으로 예상 종료 일정을 도출하는 등 일정 관련 리스크를 선제적으로 방어할 수 있도록 합니다. 또한 산출물을 분석하여 개발자별 할당된 업무량을 도축하고 업무 난이도 분석결과 및 진척률을 시각적으로 제공함으로써 프로젝트에 투입된 인원이 수행한 성과에 대해 객관적인 지표를 제공하고 효율적인 인적자원관리를 할 수 있도록 도움을 줍니다. 마지막으로 커밋 시 코드 리뷰를 제공함으로써 프로젝트 표준 준수와 코드 품질 향상을 실현 할 수 있도록 합니다.

고객 Pain point
- 진척도 관리의 불확실성: 개발자가 전달하는 주관적인 보고에 의존하게 되어 실제 코드 진척 상태를 PL이 실시간으로 확인하기 어려움
- 개발자별 업무 편중: 특정 개발자에게 업무가 과도하게 쏠려 있는지 등 개발자 자원 현황을 객관적으로 파악하기 어려움
- 리뷰 리소스 부족: 파트 리더의 시간은 한정되어 있어, 모든 커밋에 대해 상세한 코드 리뷰를 진행하기 어려움

핵심 기능
- 개발 진척도 모니터링
- 개발자별 업무량의 시각화
- 개발자별 업무 진행도의 시각화
- 개발자별 업무 난이도의 시각화
- AI 코드리뷰

사용 목적
- 개발 진척의 정량화를 통한 일정 지연 리스크의 사전관리
- 인적자원의 효율적인 관리를 위한 지표 도출
- 표준 준수 및 코드 품질 향상

기대 효과
- 납기 준수율 향상 : AI가 개발 데이터(커밋 등)를 실시간 분석하여 '지연 가능성'을 사전에 경고함으로써, 프로젝트에 발생하는 병목 현상을 방지
- 프로젝트 가시성 확보: 업무별 진행현황을 시각화하여 프로젝트 가시성을 확보
- 인력 운영 최적화 : 업무량 및 진행현황을 시각화 하여 업무 부하를 모니터링하고 특정 인력에게 업무 쏠림 현상 방지
- 코드 품질 상향평준화 : AI가 실시간으로 높은 수준으 리뷰를 제공함으로써 주니어 개발자의 성장을 돕고 코드 품질을 확보함

사업 적용 구분
- Prototyping/PoC

고객가치 구현 (정량적 수치로 작성)
- 운영 효율화 (리스크 축소 등): 리스크 사전 관리를 통해 추가로 투입되는 자원 감소 (프로젝트당 1MM 이상 예상)
- 생상선 향상 (AI코드 리뷰로 생산성 향상에 기여)
```
